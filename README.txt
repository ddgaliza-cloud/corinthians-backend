
COMO EXECUTAR O PROJETO

1. Instale o Node.js
https://nodejs.org

2. Extraia a pasta do projeto

3. Abra a pasta no VSCode

4. Abra o terminal e execute:

npm install

5. Depois execute:

npm run dev

6. Acesse:
http://localhost:3333

ROTAS:

GET /jogadores
POST /jogadores
PUT /jogadores/:id
DELETE /jogadores/:id
